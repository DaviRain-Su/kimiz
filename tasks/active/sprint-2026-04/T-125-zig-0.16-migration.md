## Research

### Completed Research
- [x] 已验证utils wrappers正确性 (generator.zig测试通过)
- [x] 已识别所有std.fs.cwd()用法
- [x] 已确认迁移模式：
  - `std.fs.cwd().openFile()` → `utils.readFileAlloc()` (读取)
  - `std.fs.cwd().createFile()` → `utils.writeFile()` (写入)
  - `std.fs.cwd().openDir()` → `utils.openDir()` (目录迭代)
  - `std.fs.cwd().readFileAlloc()` → `utils.readFileAlloc()`
  - `std.fs.cwd().makeDir()` → `utils.makeDirRecursive()`
  - `std.fs.cwd().access()` → `utils.fileExists()`

### Migration Patterns
参考 `src/skills/generator.zig` 中的成功模式：
```zig
// OLD (Zig 0.15):
const file = try std.fs.cwd().openFile("path", .{});
defer file.close();
const content = try file.readToEndAlloc(allocator, max_size);

// NEW (Zig 0.16):
const content = try utils.readFileAlloc(allocator, "path", max_size);
defer allocator.free(content);
```
