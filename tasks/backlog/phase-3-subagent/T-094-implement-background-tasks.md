### T-094: implement-background-tasks
**状态**: pending
**优先级**: P1
**创建**: 2026-04-05
**预计耗时**: 14h

**描述**:
实现后台任务机制，允许用户将长时间运行的任务放到后台执行。

kimi-cli 支持后台任务，用户可以继续前台交互。本任务需要：
1. 设计后台任务队列和任务状态跟踪（运行中/完成/失败）
2. 使用 Zig 异步运行时（libxev 或线程池）实现任务并发
3. 添加 `/bg` 或类似命令将当前任务放入后台
4. 实现 `/jobs` 命令查看后台任务列表和状态
5. 任务完成后通过文字/TTS 等方式通知用户（可选）
6. 支持运行中发消息调整（steer）后台任务的行为
