### T-121: global-context-injection-for-subagents
**状态**: pending
**优先级**: P0
**创建**: 2026-04-05
**预计耗时**: 14h

**描述**:
设计全局上下文注入机制，确保每个子代理都能获得统一的项目级基础知识。

Swarm 只实现了文件系统隔离，但没有解决语义上下文隔离的问题。本任务需要：
1. 定义项目级全局上下文来源：SOUL.md / ARCHITECTURE.md / 编码规范 / Skill Registry
2. 在子代理启动时，自动将全局上下文注入到其 system prompt 或工作内存中
3. 设计上下文缓存机制，避免每次启动子代理都重复读取和解析
4. 支持上下文版本控制：当全局文档更新时，子代理能感知到变更
5. 确保注入的上下文不会与子代理的私有会话历史混淆

参考文档: docs/SWARM-PENBERG-ANALYSIS.md
